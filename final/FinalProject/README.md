CSE210 Final Project Designer Activity – Joshua Walker
Overview: As a LEGO enthusiast, my LEGO collection has grown to be incredibly hard to manage. This program helps establish a personal inventory and selling inventory -- one collection that stays with me, and another with sets I am trying to sell. Records of transactions and revenue are recorded in the program, and details for each LEGO Asset is well outlined.

Program Requirements:
Allow users to create different LEGO assets (Minifigure, Full Sets, Incomplete Sets)
Allow users to add LEGO assets to their inventories – whether it be their selling inventory or personal inventory.
Provide functionality for the inventory assets to be moved to other collections
Be able to view the estimated value of a collection, search for items by theme, record sellings of items, and store transactions in a list
Store/load collections in/from a file

DEMO: The best way to view this project is by loading the test.txt file that already has data to work with. Otherwise, you will spend a lot of time creating LEGO items for your inventory, and if you're not as passionate about LEGO as I am, that might get boring real fast.
